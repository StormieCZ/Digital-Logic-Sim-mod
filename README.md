# Features<br>
Feature | Description | Limits | Issues | Goodies
:-: | :-: | :-: | :-: | :-:
Websockets | Allows for connection to the internet | Limited to the local network | None *(so far)* | *Unlimited* amount of websockets
Speakers | Allows playing audio frequencies | None so far | Audio glitches may occur (67% Chance) | You can now have bad apple with sound
16 & 32 bit pins | Adds 16 & 32 bit pins<sub> *(duh)*</sub> | None *(so far)* | Very **thic** wires and messy splitters | Ehh more data i guess
HZ Clock | A little clock that can be sped up | Limited by your pc | None *(so far)* | SPEEEEEEEEEEED
RNG Chip | When powered starts going haywire, when then unpowered chooses if output is on or off based on the weather (kidding) | None | None | _Gambling?_
