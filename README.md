# Features<br>
Feature | Description | Limits | Issues | Goodies
:-: | :-: | :-: | :-: | :-:
Websockets | Allows for connection to the internet | Limited to the local network | None *(so far)* | *Unlimited* amount of websockets
Speakers | Allows playing audio frequencies | None so far | None *(so far)* | You can now have bad apple with sound
16 & 32 bit pins | Adds 16 & 32 bit pins<sub> *(duh)*</sub> | None *(so far)* | Very **thic** wires and messy splitters | Ehh more data i guess
HZ Clock | A little clock that can be sped up | Limited by your pc | Crashes when going <1 ticks per second | SPEEEEEEEEEEED
RNG Chip | When power starts going haywire, when then unpowered chooses if output is on or off based on the weather (kidding) | None | None | _Gambling?_


> [!CAUTION]
> Crashes are always a possibility and can lead to loss of progress, when chip / project is unsaved! <br><sub>You have been warned.
